# PWA Icons

To complete the PWA setup, add icon files to this directory:

## Required Icons for PWA Manifest:
- icon-72x72.png
- icon-96x96.png  
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png
- icon-32x32.png (favicon)
- icon-16x16.png (favicon)

## How to Create Icons:

1. **Design your icon** (512x512px recommended)
2. **Use an online PWA icon generator** like:
   - https://www.pwabuilder.com/imageGenerator
   - https://realfavicongenerator.net/
   - https://favicon.io/favicon-generator/

3. **Or use any image editor** to resize your logo to the required dimensions

## Icon Guidelines:
- Use PNG format
- Ensure good contrast and readability at small sizes
- Consider both light and dark backgrounds
- Make it recognizable as your brand

The current manifest.json is configured to use these icon sizes. Replace this README with actual icon files when ready.
